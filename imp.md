# Project: Stock Sentiment Analyzer (Microservices)

## Tech Stack
- **Backend:** Python (Flask), `yfinance` (Data), `vaderSentiment` (ML).
- **Frontend:** React (Vite), TailwindCSS (Dark Mode), Recharts (Dual-axis charts).
- **Infra:** Docker Compose (Orchestration).

## Global Rules for Agents
1. **No Placeholders:** Never write `// code goes here`. Write the full implementation.
2. **Error Handling:** All API calls must be wrapped in `try/except` blocks.
3. **Environment:** Never hardcode ports or URLs. Use `os.getenv` or `import.meta.env`.
4. **Style:** Use functional React components with Hooks. Use Python PEP8 standards.
